WHAT HAPPENS WHEN YOU RUN npm install
================================================================================

The command will:
1. Read package.json to see what's needed
2. Download all required packages
3. Install them in node_modules folder
4. Create/update package-lock.json
5. Complete with "added XXX packages" message

This takes 2-5 minutes depending on internet speed.

================================================================================
WHAT HAPPENS WHEN YOU RUN npm run dev
================================================================================

The command will:
1. Start Next.js development server
2. Compile all files
3. Start watching for changes
4. Open server on http://localhost:3000
5. Show you the URL where the app is running

The server stays running until you press Ctrl+C.

================================================================================
DATABASE MIGRATION PROCESS
================================================================================

Three SQL files run in order:

1. 01_create_tables.sql
   - Creates all database tables
   - Sets up structure
   - Creates indexes

2. 02_insert_sample_data.sql
   - Currently empty (no demo data)
   - You'll add data manually

3. add-missing-tables.sql
   - Creates additional tables
   - Sets up user_id columns
   - Fixes any missing pieces

DB_USER ..................... MySQL username (root)
DB_PASSWORD ................. MySQL password (what you set during MySQL install)
DB_NAME ..................... Database name (skillauro)
EMAIL_SERVICE ............... Gmail or other (gmail)
EMAIL_USER .................. Your email (your_email@gmail.com)
EMAIL_PASSWORD .............. Gmail App Password (16-character password from Gmail)
RAZORPAY_KEY_ID ............ From Razorpay dashboard
RAZORPAY_KEY_SECRET ........ From Razorpay dashboard

NEXTAUTH_URL ............... Application URL (http://localhost:3000)
