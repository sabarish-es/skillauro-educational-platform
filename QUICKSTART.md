# Skillauro Platform - Quick Start Guide

Get up and running in 5 minutes!

## 1. Extract & Open Project

Extract the ZIP file and open the folder in VS Code.

## 2. Install & Run

Open terminal (Ctrl + `) and run:

```bash
npm install
npm run dev
```

## 3. Open Browser

Visit: `http://localhost:3000`

## Common Issues

### Port 3000 already in use?
```bash
npm run dev -- -p 3001
```

### Can't login?
- Copy exact email & password from above
- Make sure to select correct role
- Check browser console (F12) for errors

### Page won't load?
- Clear browser cache (Ctrl + Shift + Delete)
- Restart dev server (Ctrl + C, then `npm run dev`)
