# Skillauro Platform - Complete Features List

## Platform Overview

Skillauro is a comprehensive online learning management system with three distinct user roles, each with specialized functionality.

---

## Admin Dashboard Features

### Dashboard Overview
- Total statistics (Students, Faculties, Courses, Revenue)
- Quick action buttons
- Recent activity feed
- System health status

### Faculty Management
- **Create Faculty**: Add new faculty members with:
  - Full name and email
  - Department/Subject specialization
  - Phone number
  - Automatic password generation
  - Role assignment

- **Edit Faculty**: Modify faculty details
  - Update contact information
  - Change department assignment
  - Reset password

- **Delete Faculty**: Remove faculty with confirmation
  - Archive option
  - Activity log tracking

- **View Faculty List**: Complete faculty directory
  - Search by name/email
  - Filter by department
  - Sort by various columns
  - Export faculty data (future)

### Student Management
- **Create Student**: Add new students with:
  - Full name, email, phone
  - Student ID generation
  - Course assignment
  - Parent/Guardian contact
  - Emergency contact

- **Edit Student**: Update student information
  - Enrollment status
  - Course changes
  - Contact information

- **Delete Student**: Remove student records
  - Archive option
  - Grade preservation

- **View Student List**: Complete student directory
  - Search by name/ID/email
  - Filter by enrollment status
  - Sort by enrollment date
  - Bulk operations (future)

### Course Management
- **Create Course**: Set up new courses
  - Course name and code
  - Description and curriculum
  - Faculty assignment
  - Duration and schedule
  - Capacity/maximum students
  - Fee amount

- **Edit Course**: Modify course details
  - Update schedule
  - Change faculty
  - Adjust fees
  - Update curriculum

- **Delete Course**: Remove courses from system
  - Archive option
  - Refund management

- **View Courses**: Complete course catalog
  - Search by name/code
  - Filter by status/faculty
  - View enrolled students
  - See course statistics

### Fee Management
- **Payment Records**: Track all payments
  - Payment ID and date
  - Student name and amount
  - Payment method (Razorpay)
  - Transaction status
  - Search and filter

- **Payment Status**: Monitor payment status
  - Pending payments
  - Completed payments
  - Failed payments
  - Refunds

- **Generate Reports**: Financial reporting
  - Daily/Monthly revenue
  - Payment reconciliation
  - Student payment status report

### Notifications
- **Send Notifications**: Broadcast messages to:
  - All students
  - Specific student groups
  - Faculties
  - Specific individuals

- **Notification Types**:
  - Course updates
  - Fee reminders
  - Class schedule changes
  - System announcements
  - Important notices

- **Notification History**: View all sent notifications
  - Search by date/recipient
  - View delivery status
  - Edit/resend notifications
